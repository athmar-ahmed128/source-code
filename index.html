1. What is Source Control? Source Control (or Version Control) is the practice of tracking and managing changes to software code.
2. Why do we need Source Control? We need it to keep a history of changes, collaborate with other developers without overwriting their work, and easily revert to previous versions if errors occur.
3. What is Git? Git is a distributed version control system that tracks changes in any set of computer files, usually used for software development.
4. What is GitHub? GitHub is a cloud-based hosting service that lets you manage Git repositories and collaborate on projects from anywhere.
