Source Control Assignment

Student Name: Athmar Ahmed
1. What is Source Control? It is the practice of tracking and managing changes to software code to maintain a history of edits.
2. Why do we need Source Control? To collaborate effectively, prevent code loss, and revert to previous versions if needed.
3. What is Git? Git is a distributed version control system used to track changes in source code during software development.
4. What is GitHub? GitHub is a cloud-based platform that hosts Git repositories, allowing for easy collaboration and sharing.
